## Quick Capture (Browser DevTools)

1. Open `http://localhost:5173`
2. Press `F12` > Device Toolbar > Set viewport to `1400 x 900`
3. Navigate to each page and use `Ctrl+Shift+P` > "Capture full size screenshot"
4. Rename and place files in this directory
